# Customer Churn Analysis – Telco

# Objective of the Analysis

The main objective of this analysis is to understand **why some customers leave the platform** and to identify the variables most associated with churn.

The analysis will specifically aim to:

* identify the customer profiles most likely to churn
* understand how long customers generally remain active
* analyze the impact of financial and contractual factors
* identify the variables most associated with churn
* propose concrete actions to improve customer retention


# Dataset Overview

The dataset used in this project contains information about subscriptions of customers from a telecommunications company.

Each row in the dataset represents a customer and includes various information such as:

* demographic characteristics (gender, senior status)
* subscribed services (internet, phone, streaming)
* contractual information (contract type, tenure)
* billing information (monthly charges, payment method)

The main variable studied in this project is **Churn**, which indicates whether a customer has left the platform or not.

* **Yes**: the customer has left the service
* **No**: the customer remains subscribed


# Analysis Questions

To achieve the project objectives, the analysis will focus on the following questions:

1. What proportion of customers leave the platform?
2. Which customer segments are most likely to churn?
3. How long are customers generally active?
4. Which financial and contractual factors are associated with churn?
5. What actions could improve customer retention?


# Analysis Approach

To answer these questions, the analysis will follow several steps:

1. **Data Cleaning and Preparation**
   Check data quality and prepare the necessary variables for analysis.

2. **Exploratory Data Analysis (EDA)**
   Explore different variables to understand the overall trends in the dataset.

3. **Analysis of Factors Associated with Churn**
   Study demographic, contractual, and financial variables to identify those that most influence churn.

4. **Statistical Tests**
   Use statistical tests (such as the chi-square test) to verify whether certain relationships are significant.

5. **Interpretation and Business Recommendations**
   Transform the analysis results into concrete recommendations to improve customer retention.


# Project Importance

This analysis allows identifying the most at-risk customer segments and helps the company better understand the factors influencing churn.

The results can be used to:

* improve retention strategies
* target at-risk customers
* optimize offered products and services

By identifying the causes of churn, the company can implement appropriate actions to **reduce customer departures and improve overall satisfaction**.
